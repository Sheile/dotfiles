Welcome to my dotfiles.

This repository contains my dotfiles(.bashrc, .vimrc, etc.) and simple script to use these dotfiles easily.

If you want to use these dotfiles, execute below commands and customize for yourself.

$ cd ~
$ git clone https://Sheile@github.com/Sheile/dotfiles.git
$ cd dotfiles
$ ./applydotfiles.sh

Install vim scripts.

I'm newbie vimmer. Some vim plugins are managed by NeoBundle. You should execute below command on vim to use all vim plugins.

$ vim
:NeoBundleInstall
